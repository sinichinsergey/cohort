# Когортный анализ

## Описание
Когортный анализ номенклатуры.

## Решенные задачи
- Провести когортный анализ номенклатуры.

## Сущности
- date - первый день месяца
- lvl_5 - номенклатура
- card_id - ключ покупателя
- checks - количество уникальных чеков

## Технологии
- Python 3.10
  - pandas
  - matplotlib
  - seaborn
